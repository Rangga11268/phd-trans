## âœ¨ Fitur Unggulan

### ðŸŽ¨ Desain "Electric Royal"

- **Floating Media Portal**: Video hero tidak lagi menjadi background berat, melainkan elemen 3D mengambang yang interaktif.
- **Dynamic Scramble Text**: Efek teks _decoding_ ala cyber-punk yang tetap mempertahankan aksesibilitas.
- **Glassmorphism 2.0**: Card transparan dengan _noise texture_ halus dan border gradasi neon.

### ðŸš€ Optimasi Performa Ekstrem (Terbaru)

- **Map Facade**: Peta Google Maps hanya dimuat saat user berinteraksi (Klik/Hover), menghemat ~300KB JS awal.
- **Smart Image Loading**: Menggunakan atribut `sizes` responsif pada semua gambar armada, menghemat bandwidth data hingga 70% di mobile.
- **GPU Acceleration**: Semua animasi berat (hover, parallax) menggunakan `will-change-transform` untuk 60FPS yang mulus.
- **Zero Layout Shift**: Penggunaan font monospace pada angka dinamis mencegah halaman "bergetar" saat loading.

### ðŸšŒ Showcase Armada Interaktif

- **Modal Fasilitas**: Detail fasilitas bus (Karaoke, AC, Kursi) ditampilkan dalam modal _carousel_ yang imersif tanpa perlu pindah halaman.
- **Fleet Grid**: Grid responsif yang menampilkan spesifikasi "Big Bus" dan "Medium Bus" dengan jelas.

## ðŸ›  Teknologi Utama

Project ini dibangun dengan stack teknologi modern untuk menjamin kecepatan dan skalabilitas.

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router, Server Components)
- **Bahasa**: [TypeScript](https://www.typescriptlang.org/) (Strict Mode)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) + Custom Design Tokens
- **Animasi**: [Framer Motion](https://www.framer.com/motion/) (Complex Gestures & Transitions)
- **Media**: Optimized Video & Next/Image
